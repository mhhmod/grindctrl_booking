Install

One Script Tag.
Any Website.

Drop the GRINDCTRL widget on Shopify, WordPress, React, Vue, or plain HTML. One script. Zero dependencies. Full AI support on every page.

code Your Embed Snippet 
<!-- GRINDCTRL Support Widget -->
<script src="https://cdn.grindctrl.com/grindctrl-support.js"></script>
<script>
  GrindctrlSupport.init({
    embedKey: 'YOUR_EMBED_KEY',
    domain: window.location.hostname,
  });
</script>
